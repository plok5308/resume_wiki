# sources/resumes

과거에 작성한 자기소개서 원본을 그대로 보관한다.

## 규칙
- 절대 수정하지 않는다 (원본 보존)
- 파일명: `{YYYY}-{회사}-{직무}.txt` 또는 `.md` (예: `2024-naver-backend.md`)
- 각 파일 상단에 메타 주석으로 지원 시점/회사/직무/결과(서류합/탈락 등)를 남겨두면 wiki 증류 시 유용하다

## 메타 예시
```
<!--
date: 2024-09
company: Naver
role: Backend Engineer
result: 서류 합격
-->
```
