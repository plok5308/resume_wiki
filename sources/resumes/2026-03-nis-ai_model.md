대규모 AI 모델은 연구 성능만으로 완성되지 않으며, 데이터 구축·학습·최적화·배포 전 주기를 구현할 수 있어야 합니다. 저는 10년 이상 딥러닝 모델을 연구·개발해 왔고, 최근에는 10명 이상의 AI 연구인력을 이끄는 팀장으로서 차량용 AI Agent 및 LLM 계열 모델 개발과 상용화를 주도하였습니다. 모델 연구에 그치지 않고 실제 서비스까지 성공시킨 점들이 제 강점입니다. 특히 데이터 구축부터 모델 학습, 최적화, 배포까지 전 과정을 총괄하며, 폐쇄형 환경에서도 활용 가능한 대규모 AI 모델 구축 역량을 축적하였습니다.

## 1. 차세대 차량용 Vision Language Model 개발
주제 - 멀티모달 기반 차량용 AI Agent 모델 구축
내용요지 - Gleo Multimodal 팀장으로 주행 상황 VQA 및 visual description 기능을 개발하였습니다. 데이터 구축 및 정제, 단계별 학습 (SFT, DPO 등) 고도화를 통해 차량 환경용 Vision-Language Model 성능 향상을 주도하였습니다.
기여도 - 팀 리딩 및 개발 총괄

## 2. LLM 기반 차세대 TTS 시스템 개발 및 상용화
주제 - 대규모 생성모델의 실서비스 최적화
내용요지 - LLM 기반 TTS의 반복, 누락 오류와 느린 추론 문제를 해결하기 위해 audio tokenizer, repetition penalty, multi-token prediction 등을 적용하였습니다. 이를 통해 생성 품질과 추론 속도를 개선하고, 다국어 차량 서비스 상용화를 주도하였습니다.
기여도 - 팀장, 모델 최적화 및 상용화 총괄

## 3. 이미지 조작 탐지 기술 개발
주제 - 데이터 기반 비전모델 연구
내용요지 - 실제 이미지 분포를 분석하여 JPEG 압축 특성을 반영한 조작 탐지 모델을 개발하였습니다. 해당 연구는 ECCV 2018 제1저자 논문으로 채택되었습니다. 2018년에는  과학기술정보통신부가 주관한 AI R&D Challenge에서 8인 팀을 이끌어 종합 3위 및 3억원 규모 과제 수주 성과를 거두었습니다.
기여도 - 제1저자, 연구 총괄 및 팀 리딩
