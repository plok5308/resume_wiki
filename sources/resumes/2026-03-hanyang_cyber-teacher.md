저의 가장 큰 장점은 인공지능 분야에서 다양한 경험을 했고, 각 경험마다 큰 성취를 이뤄냈다는 점 입니다. 박사과정 시절에는 이미지 조작 탐지 연구에 대해 깊게 몰두한 뒤, 실용적인 Double JPEG Detection 기반의 이미지 조작 탐지 기술을 제안하였습니다. 그 결과 연구실 최초로 Computer Vision 분야 최상위 학회인 ECCV 2018 에 저서를 게재하고 발표하는 경험을 하였습니다. 또한 연구실 동료들과 함께 팀을 만들고, 제가 그 팀을 이끌어 과학기술정보통신부가 주관한 "2018년도 인공지능 R&D 챌린지" 에서 입상한 경험이 있습니다. 국내의 우수한 연구실, 연구소, 기업이 참여한 대회에서 종합 3등을 달성하였고, 3억원의 추가 연구비를 수주하였습니다. 

삼성전자 재직 당시에는 스마트폰의 핵심 두뇌인 AP (Exynos) 를 제작하는 사업부에서 일을 했습니다. NPU를 직접 설계하는 팀에서 NPU의 동작을 검증하는 일을 맡아서 했었으며, ISP 를 자동으로 튜닝하는 기술 또한 개발하였습니다. ISP 튜닝 기술은 삼성 그룹 전체를 대상으로 심사하는 삼성그룹 논문상 "은상" 을 수상하기도 하였습니다. 

현대자동차와 42dot에서는 다양한 음성 분야의 기술들을 직접 확보하여 서비스로 제공하였습니다. Multi speaker detection, Streaming speech recognition 기술을 주도적으로 개발하였으며, 해당 기술들은 여전히 다양한 차종에서 서비스되고 있습니다. 최근에는 다양한 상황에 맞게 사람처럼 자연스럽게 말을하는 NaturalTTS 기술 개발을 주도하였으며, 이를 인정받아 audio generation 팀을 공식적으로 이끄는 리더에 임명되었습니다. 제가 주도해서 개발한 NaturalTTS 기술은 곧 상용화되어 다양한 현대자동차 그룹의 차량에 적용될 예정입니다. 다양한 프로젝트 성공을 뒤로 하고, 새로운 도전을 위해서 현대자동차그룹에서의 경력은 마무리 진행중입니다. 

이처럼 다양한 프로젝트를 수행하면서 저는 오픈소스 소프트웨어가 단순한 개발 도구가 아니라, 협업과 생산성을 극대화하는 핵심 인프라라는 것을 체감하였습니다. 실제 업무에서는 Git 기반 버전 관리, 코드 리뷰, 이슈 관리 등 오픈소스 생태계 중심의 협업 방식이 필수적으로 요구되며, 이러한 경험은 자연스럽게 실무 역량으로 이어진다고 생각합니다.
저는 이러한 실무 경험을 바탕으로 학생들이 단순히 오픈소스를 사용하는 수준을 넘어, 실제 개발 환경과 유사한 협업 경험을 할 수 있도록 수업을 설계하고자 합니다. 구체적으로 GitHub 기반 Pull Request 및 Issue 관리, 문서화(Notion), 클라우드 환경 활용 등을 실습 중심으로 구성하여, 학생들이 직접 협업 과정을 경험하도록 유도할 계획입니다. 이러한 교육을 통해 학생들이 단순한 학습자를 넘어, 실제 오픈소스 개발 생태계에 참여할 수 있는 실질적인 역량을 갖춘 인재로 성장하도록 기여하고 싶습니다.
